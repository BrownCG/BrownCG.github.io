SunsetPi



This is a Wifi-connected timelapse camera that gathers sunrise or sunset times for your area via openweathermaps.org API calls, then sleeps until 20 minutes before sunset. It then takes a series of still images and stitches them into an .mp4 file, which is then emailed to the user. Finally, it cleans up all the files it has created and goes back to sleep waiting for the next day. Because of the hardware and various account setups this requires, I am not currently sharing the code for this project. (Note image quality was sacrificed to convert to a .gif file for this page.
